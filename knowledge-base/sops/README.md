# 📝 Procedimentos Operacionais Padrão (SOPs)

Standard Operating Procedures (SOPs) são o coração de uma equipe de TI eficiente. Eles garantem que qualquer técnico execute uma tarefa exatamente da mesma maneira, garantindo previsibilidade, auditoria e facilidade de manutenção.

| SOP | Aplicação Prática |
|-----|-------------------|
| [SOP 001: Onboarding de Novo Computador](./sop-001-onboarding.md) | O que instalar, quais updates fazer e como vincular a máquina corporativa ao domínio AD. |
| [SOP 002: Reset de Senhas e MFA](./sop-002-senhas-mfa.md) | Fluxo seguro para resetar senhas no AD, forçar troca no login e vincular o dispositivo Autenticador (Token). |
| [SOP 003: Instalação de Impressoras de Rede](./sop-003-impressoras.md) | Configuração de drivers via Print Server e IP fixo para evitar perda de comunicação por DHCP. |
| [SOP 004: Backup Preventivo antes de Formatação](./sop-004-backup.md) | Lista de diretórios cruciais (Downloads, Documentos, Desktop, PSTs de E-mail) que não podem ser esquecidos. |
